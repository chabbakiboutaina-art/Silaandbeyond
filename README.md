# Sila & Beyond — Amour Complice

Site e-commerce premium pour **Sila & Beyond**, vendant le **Amour Complice Couple Planner**.

## Structure du projet

```
sila-beyond/
├── index.html          # Site complet (5 pages en SPA)
├── src/
│   ├── style.css       # Tous les styles
│   └── main.js         # JavaScript (navigation, panier, FAQ, formulaire)
├── public/
│   ├── hero.jpg        # Photo flatlay principale
│   ├── planner.jpg     # Photo du planner ouvert
│   └── logo.png        # Logo Sila & Beyond
└── README.md
```

## Pages incluses

- **Accueil** — Hero, trust bar, bénéfices, contenu du planner, section cadeau, CTA final
- **Produit** — Galerie, détails, bouton Ajouter au panier
- **À propos** — Histoire de la marque, valeurs
- **FAQ** — Accordion interactif
- **Contact** — Formulaire avec confirmation

## Fonctionnalités

- Navigation SPA (Single Page Application) sans rechargement
- Panier latéral avec gestion des quantités
- Popup newsletter (1 fois par session)
- Menu hamburger mobile
- FAQ accordion
- Formulaire de contact avec message de confirmation
- Navbar sticky avec ombre au scroll
- Animations d'entrée (fadeUp)
- Entièrement responsive (mobile, tablette, desktop)

## Utilisation

Ouvrir `index.html` directement dans un navigateur.  
Aucune dépendance, aucun build requis.

## Déploiement GitHub Pages

1. Pusher ce dossier sur GitHub
2. Aller dans **Settings → Pages**
3. Sélectionner **Branch: main** et **Folder: / (root)**
4. Le site sera disponible sur `https://[username].github.io/[repo]/`

## Design

- **Typographies** : Cormorant Garamond + Jost + Dancing Script (Google Fonts)
- **Palette** : Crème `#FAFAF8` · Ivoire `#F4F1EC` · Noir `#181614` · Or `#B89A6A`
- **Style** : Minimaliste premium, éditorial, romantique — inspiré Shopify luxe
